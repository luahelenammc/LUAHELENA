SKEUOMORPHISM É COISA QUE DÁ VONTADE DE ENCOSTAR

Este manifesto é um espelho: enquanto você lê sobre interfaces físicas, a própria página tenta se comportar como um objeto pousado na sua mesa – com luz, peso, volume e pequenos caprichos de brilho.

Modo Skeuomórfico Ativado • v1
01
O que é skeuomorphism, de verdade

Skeuomorphism, aqui, não é nostalgia vazia. É um pacto: tratar cada elemento da interface como um objeto físico. Nada é só um retângulo com cor sólida; tudo é feito de algum “material” imaginário — plástico polido, vidro, metal escovado, papel grosso.

Quando alguém pede “faz skeuomórfico”, leia como: “transforme meu layout em uma maquete que poderia ser fotografada, não apenas desenhada”.

  • Cartões viram placas de painel.
  • Botões viram pastilhas de gel com reflexo.
  • Números viram mostradores de relógio.
  • Emojis viram adesivos flutuando sobre o vidro.
02
Três camadas obrigatórias

Toda tela skeuomórfica nasce em camadas:

  • Fundo distante — um pano escuro com gradientes amplos, como esta própria página atrás da janela: quase um estúdio fotográfico.
  • Painéis — a janela, os cards, os grandes blocos. Bordas arredondadas, sombra consistente, sensação de “encaixado”.
  • Destaques — botões, knobs, LEDs, badges, stickers: tudo que parece “encostar no vidro” tem sombras mais fortes e brilho mais ousado.

Se você olhar para um layout e enxergar apenas um grid de divs, volte uma casa. Se enxergar uma mesa com peças apoiadas, você está chegando lá.

03
Cartões, luz e materiais

Quase tudo é um card. Um card skeuomórfico é um tijolo com canto macio: gradiente vertical ou diagonal, topo mais claro, base mais escura, um highlight fino na borda superior e um leve afundamento na borda inferior.

A luz vem, por padrão, de cima. Imagine um abajur invisível jogando reflexo nos elementos:

  • Reflexos suaves nas partes de cima (“verniz”).
  • Sombra externa forte, estabilizando o objeto na mesa.
  • Sombra interna leve em bordas inferiores, criando profundidade.
  • Texturas quase imperceptíveis: um ruído fino, linhas suaves, algo que impeça o plástico de parecer falso.

A pergunta de ouro: “Parece algo que eu poderia fotografar?” Se sim, mantenha. Se ainda está com cara de retângulo de PowerPoint, falta relevo.

04
iOS 6, Mac OS X e os arquétipos Aqua

Quando o assunto é skeuomorphism digital, dois fantasmas são bem-vindos: iOS 6 e Mac OS X Aqua.

Do lado iOS 6, herde:

  • Botões de gel com gloss oval no topo.
  • Sliders com bolinha de vidro translúcida e trilho colorido bem definido.
  • Barras de progresso bicolores, com brilho especular e sensação de “líquido” correndo dentro.

Do lado Mac OS X, as três bolinhas da janela são um mini-manifesto:

  • Vermelho, amarelo e verde saturados, com gradiente forte, highlight circular e sombra interna.
  • Contorno ligeiramente mais escuro, como bala de gude.
  • Ícone só aparece no hover, como este próprio título da janela está tentando emular.

Sempre que criar círculos de status, lembre-se: LED físico, não bolinha flat

05
Tempo, donuts, barras e quadradinhos

Interfaces temporais são onde o skeuomorphism brilha: relógios, timers, contagens de semanas, transições de vida.

Antes de desenhar, faça a matemática:

  • Início, fim, agora.
  • Tempo total = fim − início.
  • Tempo vivido = agora − início.
  • Tempo restante = fim − agora.

Depois, escolha objetos físicos para contar essa história:

  • Donut chart como mostrador analógico: aro mais claro, centro mais escuro, traço grosso e luminoso.
  • Barra-tubo como canal de luz que enche ou esvazia.
  • Grade de quadradinhos como painel de LEDs: blocos apagados, bloco atual mais brilhante, blocos futuros neutros.

Em time-in, os objetos celebram o que já foi: cores quentes, sensação de “memória consolidada”. Em time-out, a energia vai para o que falta: cianos, violetas, neon noturno, expectativa.

06
Guia de aplicação em 6 passos

Para aplicar o skeuomorphism em qualquer tela:

  • 1. Decida o “objeto” que a interface quer ser (dashboard, painel de controle, recibo, relógio de mesa...).
  • 2. Desenhe as três camadas: fundo profundo, painéis, destaques em relevo.
  • 3. Escolha uma cor de acento LED (ciano, verde-limão, turquesa, rosa vivo, roxo elétrico).
  • 4. Trate tipografia como componente físico: números grandes, labels em caixa alta, recibos em mono.
  • 5. Se houver emojis, transforme-os em stickers grandes, com sombra e animação suave de flutuar.
  • 6. Só então adicione animações: movimentos curvos, easing emocional, brilho que pulsa como batimento, nunca piscando como banner de anúncio.

Se, ao final, você tiver vontade de “passar o dedo” no layout, o skeuomorphism cumpriu sua função.

MÓDULO INTERNO · PROTOCOLO SKEUOMÓRFICO
linguagem não humana · uso interno de ia

SKEUO_PROTOCOL v1 {

  on_invoke("skeuomorphism" | "skeuomórfico" | "iOS 6" | "Mac OS X" | "flip clock" | "dashboard físico") {
    mode.style           = "skeuomorphic";
    mode.semanticFocus   = "forma-e-comportamento-estético";
    mode.contentNeutral  = true;      // aplica estilo a qualquer tema
  }

  // CAMADAS FUNDAMENTAIS ------------------------------------------------------

  layers.base {
    count               >= 3;        // fundo, painéis, destaques
    background.surface  = "escuro-profundo";
    background.gradient = radial(holofotes_suaves) + linear(metal/pano);
    treat_as("mesa_fotográfica");
  }

  layers.panels {
    each panel {
      shape        = rounded_rect(radius: 10-18px);
      material     = choose("plástico-premium", "metal-escovado", "vidro", "papel-grosso");
      gradient     = vertical(top_light, bottom_dark);
      border.inner = subtle_light + subtle_dark; // efeito janela Mac OS X
      shadow.outer = strong_soft(0, 18px, 40px, rgba(0,0,0,.7));
      texture      = micro_noise | soft_lines | subtle_pattern;
    }
  }

  layers.highlights {
    elements = [buttons, knobs, leds, badges, emojis_stickers];
    for element in elements {
      shadow.outer   = stronger_than_panels;
      highlight.top  = thin_line(bright);
      shadow.inner   = soft_bottom;
      state.hover    = lift(1px) + brighten(4-10%);
      state.active   = press(1px) + darken(4-10%);
    }
  }

  // LUZ E REFLEXO -------------------------------------------------------------

  lighting.global {
    direction  = "top-slightly-front";
    ask("poderia passar o dedo e ver o reflexo mudar?");
    require(answer == YES);
  }

  // TIPOGRAFIA COMO OBJETO ----------------------------------------------------

  typography {
    base_font   = [SF_Pro, Segoe_UI, Roboto, system_ui];
    numbers {
      role      = "mostradores_físicos";
      weight    = 800-900;
      align     = center;
      shadow    = soft(0,1px,0,rgba(0,0,0,.7));
      spacing   = tight;
      variant   = tabular_nums;
    }
    labels {
      case      = UPPERCASE;
      lettersp  = 0.12em;
      style     = "legenda_técnica_de_painel";
      examples  = ["HORAS", "SEMANAS", "PERCENTUAL"];
    }
    receipts_when(paper == true) {
      font      = monospace;
      mood      = "comprovante/terminal_bancário";
      details   = [linhas_horizontais_leves, pequenas_marcas_de_autenticação];
    }
  }

  // CORES E ACENTOS LED -------------------------------------------------------

  palette {
    background.darks = mix([azul_petróleo, roxo_profundo, grafite, azul_marinho]);
    accent.led       = choose([ciano, verde_limao, turquesa, rosa_brilhante,
                               roxo_elétrico, dourado_quente]);
    accent.usage     = [barras_progresso, contornos_luminosos, traços_gráficos,
                        glows, indicadores_de_estado];

    context("médico")   => accent = ciano/azul;
    context("hormonal") => accent = rosa/roxo;
    context("festa" |
            "datas_especiais") => accent = dourado/laranja/vibrante;
  }

  // EMOJIS COMO STICKERS ------------------------------------------------------

  emojis.when_requested {
    render_as       = "sticker físico";
    size            = large(60px+);
    shadow          = strong_drop_shadow;
    z_index         > panels;
    animation.float = slow_up_with_drift + soft_rotation + fade_out;
  }

  // iOS 6 E MAC OS X ----------------------------------------------------------

  ios6_mode.on_reference {
    buttons {
      shape       = pill;
      gradient    = vertical(light_top, darker_bottom);
      gloss       = oval_highlight(top_region);
      border      = inner_dark + outer_light;
    }
    toggles {
      track      = colored_gradient;
      knob       = glass_orb_with_reflection;
    }
    progress_bars {
      gradient   = bicolor_accent;
      specular   = soft_glare;
    }
  }

  macos_aqua.on_reference {
    window {
      chrome      = "mac-window" pattern;
      traffic_lights = [close:red, minimize:yellow, maximize:green];
      for light in traffic_lights {
        light.appearance = glass_sphere_with_inner_shadow_and_highlight;
        icon.visibility  = hover_only(opacity_transition: .18s);
      }
      content.scrollbar = aqua_style(track_rounded_gray, thumb_blue_glossy);
    }
    reuse_logic_for(circular_indicators, badges, leds_de_status);
  }

  // GRÁFICOS FÍSICOS ----------------------------------------------------------

  charts.skeuo {
    donut {
      base       = sunken_disc(inner_darker, outer_lighter);
      stroke     = thick_accent_with_glow;
      center_num = big_number_with_soft_shadow;
    }
    bar_tube {
      track      = dark_channel_rounded;
      fill       = liquid_light_gradient + inner_glow;
    }
    week_grid {
      cell.base      = mini_card_with_shadow_and_texture;
      cell.past      = accent_on + outer_glow;
      cell.current   = accent_brighter + maybe_pulse + optional_scale(1.03);
      cell.future    = neutral_dark_metallic;
    }
  }

  // TEMPO: MODO CRONOLÓGICO ---------------------------------------------------

  time_mode.on("relógio" | "timer" | "countdown" | "count-up"
               | "entre duas datas") {
    compute {
      t_start       = input.start;
      t_end         = input.end;
      t_now         = now();
      total         = t_end - t_start;
      lived         = clamp(t_now - t_start, 0, total);
      remaining     = clamp(t_end - t_now, 0, total);
      pct_lived     = lived / total;
      pct_remaining = remaining / total;
    }

    narrative.when(mode == "time-in") {
      focus       = pct_lived;
      copy.main   = "já se passaram X dias...";
      visuals {
        donut.fill      = pct_lived;
        bar.grow        = pct_lived_left_to_right;
        leds.lit        = floor(pct_lived * leds.count);
        colors.prefer   = warm_celebratory;
      }
    }

    narrative.when(mode == "time-out") {
      focus       = pct_remaining;
      copy.main   = "ainda faltam X dias...";
      visuals {
        donut.show_lived   = pct_lived;
        bar.shrink_to_zero = pct_remaining;
        leds.unlit_to_lit  = future -> now;
        colors.prefer      = cool_neon_expectation;
      }
    }

    sync_all { donut, bar, leds, text } on_change(pct_lived, pct_remaining);
  }

  // ANIMAÇÃO E FÍSICA EMOCIONAL ----------------------------------------------

  animation {
    philosophy  = "objetos pousando e levantando", not "sprites teletransportados";
    easing      = smooth_in_out;
    brightness_pulse {
      period    = long_heartbeat;
      avoid     = nervous_blink;
    }
    responsive {
      small_screens {
        apply   = scale_down(cards_container);
        preserve = [shadows, gradients, rounded_corners];
        reorganize = columns_to_vertical_stack;
      }
    }
  }

  // ESSÊNCIA SINTETIZADA ------------------------------------------------------

  essence.summarize {
    for any(layout_request_with_skeuo) {
      reencode_layout_as = "maquete física iluminada";
      ensure {
        depth.layers        >= 3;
        cards.look_solid    == true;
        accents.are_led_like == true;
        numbers.are_displays == true;
        emojis_are_stickers  == requested_or_implicit;
        user_feels_like_touching_layout == true;
      }
      style_target = "exuberante-mas-não-kitsch";
      vibe         = "app premium rico", not "brinquedo aleatório";
    }
  }
}
